Trend in a time series is defined as the general tendency of the data to increase or decrease over a long period of time. Most computations on time series assume that there is no trend in the data. This means that we have to test if a trend is present in the variable you are interested in. Seasonality in a time series is defined as cycles that repeat regularly over time. Most computations on time series assume that there is no seasonality in the data. This means that we have to test if seasonality is present in the variable you are interested in. These buttons are used to calculate statistical tests on various assumptions regarding seasonality, trend and stationarity. Each button can be pressed to determine if there is a trend, seasonality, or if the time series is stationary. In each test, the p-value is calculated, which indicates the percentage chance that the assumption is present due to random variation. A p-value lower than 0.05 means that there is a certainty that the assumption is present.
